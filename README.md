# LT CURRENT TEST

## Requirements

First install Python

-   Python >= 3.12 (<https://www.python.org/>)

Then install the requirements

```bash
python3 -m pip install --upgrade pip
python3 -m pip install --upgrade -r requirements.txt
```

